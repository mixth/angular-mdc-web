Floating Action Button

A floating action button represents the primary action in an application.

References

  • Material Design guidelines: Floating Action Button
  • Material Components Web

Import

Design & API Documentation

Selector: button[mdc-fab] a[mdc-fab]

Properties Description
mini: boolean Make the fab smaller (40 x 40 pixels).
exited: boolean Animates the FAB in or out of view.
extended: boolean Modifies the FAB to wider size which includes a text label.
label: string Optional, for the text label. Applicable only for Extended FAB.
position: string Set the fab absolute position.

Valid values: 'bottomLeft' | 'bottomRight'
icon: string Optional. Apply a Material Icon.
Methods Description
toggleExited(exited?: boolean) Toggle the fab animation in and out.
focus() Focuses the button.

Selector: mdc-fab-label, [mdcFabLabel]

Optional, for the text label. Applicable only for Extended FAB.
