Icon Buttons

Icon buttons allow users to take actions, and make choices, with a single tap.

References

  • Material Design guidelines: Toggle Buttons
  • Material Components Web

Import

Design & API Documentation

Selector: button[mdcIconButton] a[mdcIconButton] mdc-icon-button

Properties Description
on: boolean Sets the toggle state to the provided value.
disabled: boolean Disables the icon button.
icon: string Optional. Set a Material Icon as a non-toggle icon.
Methods Parameters
toggle(isOn?: boolean) Toggles the icon toggle state. If an argument is given, will toggle on if true, off if false.
Events Description
change: MdcIconButtonChange Emits when the icon is toggled.

Selector: mdcIcon[mdcIconOn] mdcIconOn

Apply to an mdc-icon, and is used to indicate the toggle button icon representing the "on" icon.
