Menus

Menus display a list of choices on temporary surfaces.

Design & API Documentation

Selector: mdc-menu

Properties Description
id: string Unique Id of the menu (auto generated if not supplied).
open: boolean Opens or closes the menu.
anchorCorner: string Override the opening point of the menu. (Default: topStart)

Values: 'bottomEnd' | 'bottomStart' | 'topEnd' | 'topStart'
anchorMargin: {{'{'}}top: number, bottom: number, left: number, right : number{{'}'}} Sets the distance from the anchor point that the menu surface should be shown.
quickOpen: boolean Sets whether the menu should open and close without animation when the open/close methods are called.
anchorElement: Element | mdcMenuSurfaceAnchor Set to indicate an element the menu should be anchored to.
fixed: boolean Used to indicate that the menu is using fixed positioning.
wrapFocus: boolean Sets the list to allow the up arrow on the first element to focus the last element of the list and vice versa.
Methods Parameters
focus() Set focus to the menu.
Event Description
selected(item: MdcMenuItem, index: number) Dispatches an event that a menu item has been selected.

Selector: mdcMenuSurfaceAnchor mdc-menu-surface-anchor

Sets the anchor element used as an anchor for menu-surface positioning logic.
